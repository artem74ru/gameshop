# Изображения для жанров

Поместите сюда изображения для жанров (категорий) на главной странице.

## Формат файлов

- **Имя файла**: должно соответствовать `slug` жанра (например, `action.jpg`, `adventure.jpg`)
- **Форматы**: `.jpg`, `.png`, `.webp`
- **Рекомендуемый размер**: минимум 300x117px (соотношение сторон примерно 2.5:1)
- **Оптимальный размер**: 600x234px или больше для лучшего качества

## Примеры названий файлов

- `action.jpg` - для жанра "Action"
- `adventure.jpg` - для жанра "Adventure"
- `role-playing-games-rpg.jpg` - для жанра "Role-playing (RPG)"
- `strategy.jpg` - для жанра "Strategy"

## Как узнать slug жанра?

Slug жанра можно посмотреть в API: `/api/genres` или в консоли браузера на главной странице.

## Примечание

Если изображение для жанра не найдено, будет использован серый фон по умолчанию.
